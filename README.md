# Настройка Linux для защиты от DDoS

Простой скрипт на Python, который настраивает сеть Linux для лучшей защиты от атак.

## Что делает
1.  Смотрит, сколько на сервере памяти и ядер процессора.
2.  Подбирает подходящие настройки для защиты от DDoS.
3.  Меняет настройки ядра Linux.

## Как установить и запустить
1.  Скачайте скрипт: `wget https://github.com/VacheLorian/ddos-protection-tuner/releases/download/1.0.0/ddos-protection-tuner.py`
2.  Дайте права на запуск: `chmod +x ddos-protection-tuner.py`
3.  Запустите для проверки (ничего не поменяется):
    ```bash
    sudo python3 ddos-protection-tuner.py --dry-run
    ```
4.  Если все нравится, запустите для настройки:
    ```bash
    sudo python3 ddos-protection-tuner.py
    ```

## Основные команды
- `--dry-run` — показать, что будет сделано, но не делать.
- `--apply` — сразу применить настройки.
- `--check` — показать текущие настройки.
- `--help` — показать подсказку.

## Важно!
- Запускайте только на своих серверах или виртуальных машинах.
- Скрипт нужно запускать от имени администратора (`sudo`).
- Все изменения записываются в лог-файл в `/tmp/`.
